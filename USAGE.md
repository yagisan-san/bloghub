## 1. BlogHub とは

BlogHubは、はてなブログ・note・Notion・PDF・YouTubeなど、バラバラに存在している発信コンテンツをまとめて整理し、**ひとつの発信ホームページ**として公開できるサービスです。

### こんな方におすすめ

- 無料ブログで発信しているが、記事が時系列で流れてしまう
- note・はてなブログ・PDFなど複数の場所に発信が散らばっている
- Xプロフィールに貼れる「発信の入口」が欲しい
- WordPressに移行する前に、まず発信を整理したい

### できること

| 機能 | 内容 |
|---|---|
| コンテンツ管理 | はてなブログ・note・PDF等を一元管理 |
| グリッド表示 | カード形式で全コンテンツを一覧表示 |
| カテゴリ表示 | ジャンル別に整理して表示 |
| 読む順番（ロードマップ） | おすすめの読む順番を設定して公開 |
| ツリー構造 | 記事の親子関係を設定して階層表示 |
| 公開URL発行 | `/u/ユーザー名` のURLを発行 |
| デザイン変更 | 背景色・カード色などを自由にカスタマイズ |

## 2. アカウント登録

### 手順

**① BlogHubにアクセス**
サービスのURLをブラウザで開きます。

**② 「無料でアカウントを作る」をクリック**

**③ メールアドレスとパスワードを入力**
- パスワードは8文字以上で設定してください。

**④ ユーザー名を設定**
- 半角英数字・ハイフン・アンダースコアで3〜30文字
- 例：`yamada-taro`、`ai_blogger_2024`
- ⚠️ この名前があなたの公開URL（`/u/ユーザー名`）になります
- ⚠️ 一度設定したユーザー名は変更できません

**⑤ 表示名を入力（任意）**
- 公開ページに表示される名前です
- 日本語OK

## 3. はてなブログの連携

はてなブログを使っている方は、APIキーを使って記事を自動で取り込めます。

### APIキーの取得方法

1. はてなブログの管理画面を開く
2. 「設定」→「詳細設定」→「AtomPub」を開く
3. APIキーをコピーする

### 連携手順

**オンボーディング画面（初回登録後）：**
1. はてなID（例：`your-hatena-id`）を入力
2. ブログID（例：`yourname.hatenablog.com`）を入力
3. APIキーを入力
4. 「記事を取り込む」をクリック

**後から連携する場合：**
管理画面 → コンテンツ管理 → 「はてな記事を再取得」ボタンから同様の操作ができます。

### セキュリティについて

> 🔒 入力されたAPIキーはサーバー側で一時的に使用するだけで、**データベースには保存されません**。安心してご利用ください。

## 4. コンテンツの追加

### はてなブログ以外のコンテンツを追加する

管理画面 → **コンテンツ管理** → 種別タブの末尾にある「**＋ 追加**」をクリック。

対応している種別：
- 📝 **note**（noteの記事URL）
- 📋 **Notion**（Notionの公開ページURL）
- 📄 **PDF**（Google DriveやDropboxなどの共有リンク）
- 📊 **スプレッドシート**（Googleスプレッドシートの共有リンク）
- ▶️ **YouTube**（YouTube動画やチャンネルURL）
- 🛍️ **BOOTH**（BOOTHの商品ページURL）
- 🔗 **その他**（任意の名前をつけて登録できる）

### 「その他」の使い方

「その他」を選んだ場合は「**コンテンツ名（必須）**」欄に名前を入力してください。
この名前が公開ページのカードタイトルになります。

例：「無料テンプレート集」「外部リンクまとめ」「お問い合わせフォーム」など。

## 5. 管理画面の各機能

### コンテンツ管理

- 各種別タブで整理されて表示されます
- ⠿ マークをドラッグ＆ドロップで並び替えができます
- 👁 アイコンで表示/非表示を切り替えられます
- 🗑 削除ボタンで削除できます

### カテゴリ

各コンテンツに設定したカテゴリを一覧で確認できます。
上部のタブでコンテンツ種別を切り替えて確認できます。

### 読む順番（ロードマップ）

「読者にこの順番で読んでほしい」という順序を設定できます。
ドラッグ＆ドロップで並び替えて保存すると、公開ページの「読む順（ロードマップ）」ビューに反映されます。

### ツリー構造

記事の親子関係を設定できます。
各記事の右側の「親コンテンツを設定」ドロップダウンで、どの記事の子にするかを選べます。

例：「AIブログの始め方（親）」→「第1回：アカウント作成（子）」→「第2回：記事を書く（子）」

### デザイン設定

公開ページの見た目をカスタマイズできます。

**プリセットテーマ（6種類）：**
- 🔵 デフォルト（白・青）
- 🌙 ダーク（黒・紫）
- 🌿 ナチュラル（ベージュ・緑）
- 🌸 サクラ（ピンク）
- 🌊 オーシャン（水色）
- 🏜️ サンド（砂・オレンジ）

**カスタムカラー：** 背景色・アクセントカラー・カード色・テキスト色などを個別に設定できます。右側のプレビューでリアルタイムに確認しながら調整できます。

### ハブ設定

| 項目 | 内容 |
|---|---|
| 表示名 | 公開ページに表示される名前 |
| アイコン画像 | プロフィールアイコン（直接アップロード可） |
| カバー画像 | ページ上部の背景画像（直接アップロード可） |
| SNSリンク | X・Instagram・YouTube・メールを設定 |
| ハブタイトル | 公開ページのタイトル |
| 説明文 | タイトル下に表示される紹介文（改行対応） |
| デフォルト表示 | 公開ページを開いたときのデフォルトビュー |
| 公開ページのカード順番 | 各コンテンツカードの表示順をドラッグ＆ドロップで変更 |
| 「その他」の表示名 | 「その他」タイプのコンテンツのカードタイトルを変更 |

## 6. 公開ページの共有

### 公開URLの確認

ダッシュボードの上部にある紫のカードに公開URLが表示されています。
「開く」ボタンで確認できます。

```
https://your-domain.com/u/あなたのユーザー名
```

### 活用方法

- **Xのプロフィール**のリンク欄に貼る
- **noteのプロフィール**に記載する
- **はてなブログのプロフィール**に記載する
- **名刺・自己紹介**に使う

### 訪問者にできること

- 記事・コンテンツを閲覧する
- グリッド・カテゴリ・読む順番・ツリーを切り替えて見る
- キーワードで検索する
- 外部リンクを開く

### 訪問者にできないこと

- コンテンツの編集・削除
- 並び順の保存
- 管理画面へのアクセス（ログインしていない場合）

## 7. デザインのカスタマイズ

### カバー画像・アイコンの設定

管理画面 → **ハブ設定** → プロフィールセクション

- 「画像を選択」ボタンから直接アップロードできます（5MB以下）
- 対応形式：JPG・PNG・GIF・WebP

### SNSリンクの設定

管理画面 → **ハブ設定** → SNSリンクセクション

URLを入力して保存すると、公開ページのプロフィール欄にカードとして表示されます。

### カードの表示順変更

管理画面 → **ハブ設定** → 「公開ページのカード順番」セクション

ドラッグ＆ドロップで各コンテンツカードの表示順を変更できます。

## 8. よくある質問

**Q. 無料で使えますか？**
A. はい、完全無料でご利用いただけます。

**Q. はてなブログ以外のブログは連携できますか？**
A. V1でははてなブログのAPI連携のみ対応しています。他のブログはURLを手動で登録できます。

**Q. ユーザー名は後から変更できますか？**
A. 現在、ユーザー名は変更できません。登録時に慎重に設定してください。

**Q. 非公開にすることはできますか？**
A. 現在はすべてのハブが公開設定です。非公開機能はV2以降で対応予定です。

**Q. 記事を削除したら公開ページからも消えますか？**
A. はい、管理画面で削除または非表示にすると即座に公開ページからも消えます。

**Q. APIキーは安全ですか？**
A. APIキーはサーバー側で記事取得にのみ使用し、データベースには保存しません。安心してご利用ください。

**Q. スマートフォンでも使えますか？**
A. 公開ページ・管理画面ともにスマートフォン対応しています。
